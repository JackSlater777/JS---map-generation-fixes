Этот мод исправляет все главные баги, возникающие при генерации случайной карты.

В дополнение к этому, мод предоставляет 3 шаблона:
- Jebus Cross JS (XL) - модифицированный шаблон Jebus Cross по мотивам HotA, в котором генерация карты начинается с центра, а не зоны игрока
- Ring (XL+U) - по мотивам старого шаблона Ring. Главное отличие - присутствие охраны между зонами
- Ring Giant (G+U) - тот же, что и предыдущий, только рассчитан на игру с модом XXL
